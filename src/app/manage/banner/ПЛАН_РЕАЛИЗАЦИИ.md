## Примечания
- Баннер будет только один, поэтому API должен работать с единственным экземпляром
- Для загрузки изображений рекомендуется использовать FormData и соответствующие эндпоинты API
- Изображения могут храниться либо на сервере в папке uploads, либо в облачном хранилище (S3, Cloudinary)
- Для предпросмотра изображений использовать URL.createObjectURL или FileReader API
- Для drag-and-drop функционала можно использовать react-dropzone или нативный HTML5 API
